# Threaded Task Manager (OS Mini-Project)

A Python-based Note Taker app that demonstrates core Operating System concepts.

## OS Concepts Implemented:
* **Multi-threading:** Uses the `threading` library to handle File I/O operations in the background. This prevents the Main Process (UI) from blocking.
* **Concurrency:** Demonstrates how the CPU manages multiple threads simultaneously.
* **Persistent Storage:** Interacts with the local file system (`os_logs.txt`) to maintain state.
* **Resource Deallocation:** Implements a delete function to remove widgets and clear memory.

## Tech Stack:
* Language: Python
* Library: Tkinter (GUI), Threading (OS Logic)
